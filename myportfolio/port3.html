ผมได้ลงรายการหนึ่ง รายการชื่อว่า Bangkok Robotics Challenge 2019

ผมได้ตกลงเเข่งรายการนี้ชึ่ง ภารกิจ ของการเเข่งขันนี้คือการเดินตามเส้นเเละกลับมาจุดเดิมโดยเวลายิ่งน้อยคือเร็วซึ่งสนามเเข่งก็จะมีกันสุ่มอุปสรรค เราก็ต้องออกเเบบหุ่นยนต์ ให้ไม่ใช่เเค่เดินตามเส้นอย่างเดียว เเค่ต้องผ่านอุปสรรคได้ เเล้วเป็นช่วงปิดเทอม ผมก็ต้องนอยู่โรงเรียนได้เยอะขึ้นทำให้มีเวลาซ้อมมากขี้น ได้ทดลองอะไรได้มากขึัน ทำให้เเข่งครั้งนี้ค่อนข้าง รู้สึกว่าเตรียมตัวมาดี

วันเเข่งวันนี้ ผมก็ได้ทำการเเข่งขันเเข่งรอบเเรกผ่านไปด้วยดี ทีมเเข่งขันค่อนข้างเยอะ พอเข้ามารอบท้ายๆ ชิ้นส่วนหุ่นยนต์เกิดพังขึ้นต้องเปลี่ยน เเล้วการที่ Hardware ที่เป็นส่วนมอเตอร์พังคืออะไรที่เเย่มากเพราะ มอเตอร์เเต่ละตัวถึงจำนวนรอบจะระบุให้ชัดเจน เเต่ก็จะมี error ของอุปกรณ์ไม่เท่ากัน ซึ่งเราก็ต้องเเก้ Code ใหม่ด้วยเเละจะเเข่งรอบสุดท้ายเเล้วเพื่อจัดอันดับ 1-3 สรุปเเก้ทันเพื่อให้จบสนามได้เเต่ไม่ดีเท่ารอบเเรก พอผลการเเข่งขันออก สรุปจบที่ อันดับที่ 3 รู้สึกว่าเราทำเต็มที่เเล้ว ปัญหาอุปกรณ์พังเป็นเรื่องปกติที่เคยเกิดเเต่ไม่เคยเกิดในรอบท้ายขนาดนี้ซึ่งรอบท้ายส่วนใหญ่จะมีเวลาเเก้น้อยกว่ารอบอื่นๆ เเต่การเเข่งขันรอบนี้ต่างออกไปผมตอบตกลงที่จะเเข่งเเต่ไม่เคยอ่านว่าจบการเเข่งเเล้วได้อะไรเลยมารู้ตัวอีกทีตอนรับรางวัลบนเวทีว่ารายการนี้หาตัวเเทนประเทศไปเเข่งขันรายการ Robot Challenge 2019 ณ กรุงปักกิ่ง ประเทศจีน
